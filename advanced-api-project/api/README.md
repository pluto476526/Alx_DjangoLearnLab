# API Docs

## Endpoints

### 1. List Books
- **URL**: `/books/`
- **Method**: `GET`
- **Perms**: Public

### 2. Book Details
- **URL**: `/books/<id>/`
- **Method**: `GET`
- **Perms**: Public

### 3. Create New Book
- **URL**: `/books/create/`
- **Method**: `POST`
- **Perms**: Authenticated users

### 4. Update Book
- **URL**: `/books/<id>/update/`
- **Method**: `PUT`
- **Perms**: Authenticated users

### 5. Delete Book
- **URL**: `/books/<id>/delete/`
- **Method**: `DELETE`
- **Perms**: Authenticated users
